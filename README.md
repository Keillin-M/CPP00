# 🧩 CPP00

Your first step into modern C++ — understanding classes, objects, and the basics of object-oriented programming.

Welcome to **CPP00**, the first project of the 42 **C++ Modules**. This module introduces the foundations of **C++ syntax and OOP concepts**, transitioning from procedural C to object-oriented design.

---

## 🚀 Project Overview

**CPP00** is part of the 42 school curriculum. The goal: learn the fundamentals of **C++ programming** by implementing small programs that emphasize classes, encapsulation, and basic object behavior.

The module focuses on:

* Understanding C++ syntax and compilation
* Creating and using classes
* Managing visibility with `public` and `private`
* Writing clean and structured C++ code

---

## 📘 Exercises Overview

* **ex00 — Megaphone**
  Introduction to C++ I/O using `std::cout` and `std::string`.
  Practice with argument handling and string manipulation.

* **ex01 — PhoneBook**
  A simple contact manager demonstrating:

  * Class design
  * Encapsulation
  * Arrays of objects
  * Basic user input handling

---

## 🧠 Key Concepts Covered

* C++ compilation process
* Namespaces and `std`
* Classes and objects
* Public vs private members
* Header (`.hpp`) and source (`.cpp`) separation
* Basic input/output with `iostream`
* Code organization and readability

## 📦 How to Use

Clone the repository:

```bash
git clone https://github.com/your-username/cpp00.git
cd cpp00
```

Compile an exercise:

```bash
make
```

Run:

```bash
./megaphone "hello world"
```

Or for the PhoneBook:

```bash
./phonebook
```

## 💡 Lessons Learned

* C++ enforces structure — and that’s a good thing
* Encapsulation improves code clarity and safety
* Headers define contracts, source files implement behavior
* Clean naming and organization matter early
* OOP is about modeling behavior, not just syntax
